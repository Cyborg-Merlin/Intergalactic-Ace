Box_Collider
	int h
		~the height of this object
	int w
		~the width of this object
	int x
		~the x coordinate of the top left corner of this object
	int y
		~the y coordinate of the top left corner of this object
	int object_Affiliation
		~if another collider shares this affiliation, they will not collide.

		~send a collision message to this collider

		~this ship's collider

	func update_Position(self, x : int, y : int) -> tuple
		~Moves this object x units right and y units down, returns it's position after moving (x, y).
	
	func hit_Check(self, affiliation : int = 0) -> bool
		~Checks to see if this object has collided with any object that it isn't affiliated with. If affiliation is 0, then it will check all objects regardless of affiliation.
